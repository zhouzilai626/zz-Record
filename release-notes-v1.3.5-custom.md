## Highlights

- Adds the phone camera QR pairing UI to the source-managed app build.
- Keeps the Windows installer and GitHub release assets aligned with the current `zz-Record` custom build.
- Adds the GitHub release workflow needed for future Windows release publishing.

## Recommended Verification

- Install `zz-Record-windows-x64.exe` from this release.
- Open the app and confirm the phone camera pairing window shows a QR code.
- Run an in-app update check from an existing `1.3.4` install and confirm `1.3.5` is detected.
